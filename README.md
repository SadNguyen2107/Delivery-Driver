# Unity Delivery-Driver
My First Unity Game

A Driving Relaxing game

Control:
  Press A -> Turn Left
  Press W -> Go Forward
  Press S -> Go Downward
  Press D -> Go Right

Mission:
  Take the Package and Deliver to the Customer
